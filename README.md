How to use
1. Create a GitHub repository named `mind3`.
2. Push the files in this repo.
3. In Vercel, import the GitHub repo and deploy (Vercel will detect a Vite app).
   - Build command: `npm run build`
   - Output directory: `dist`

Local development
- Install dependencies: `npm install`
- Dev server: `npm run dev`
- Build: `npm run build`
- Preview build: `npm run preview`
